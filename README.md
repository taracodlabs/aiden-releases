## Install

**One-liner (PowerShell 5.1+):**

```powershell
irm aiden.taracod.com/install.ps1 | iex
```

**Manual installer:** download the latest `.exe` from [Releases](https://github.com/taracodlabs/aiden-releases/releases/latest), run it, and launch Aiden via Start menu or `aiden pc`.

## What is Aiden?

Aiden is a local-first Windows AI OS that:

- Runs **100% on your machine** (or any 13 cloud providers via BYOK)
- Sees your screen, reads your files, controls your computer
- Remembers everything across sessions (6-layer memory)
- Ships with 56 skills out of the box — research, trading, code, automation, Windows admin, content creation, voice
- Open source under AGPL-3.0. Auditable. Forkable.

## System requirements

- Windows 10 or 11 (64-bit)
- ~2 GB free disk space
- Internet connection for first-run setup and optional cloud providers
- Optional: NVIDIA GPU with 6+ GB VRAM for local model acceleration (GTX 1060 and above)
