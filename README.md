# aiden-releases

Binary installer releases for [Aiden](https://github.com/taracodlabs/aiden).

**Source code, issues, contributing, and discussions live at the main repo:**
https://github.com/taracodlabs/aiden

This repo only hosts release artifacts (.exe, latest.yml, blockmap) for Electron auto-update.

## Latest release

[v3.19.0 — ALIVE](https://github.com/taracodlabs/aiden-releases/releases/latest)

## Install

- **Windows:** download the .exe from latest release
- **Any platform:** `npx aiden-os` (npm)
- **PowerShell:** `irm aiden.taracod.com/install.ps1 | iex`
- **Linux/macOS:** `curl -fsSL aiden.taracod.com/install.sh | bash`
